## ✨ 主要功能

### 🤖 智能对话系统
- 与 AI 女友进行自然语言对话
- 支持 DeepSeek 和 GLM 等多种 AI 模型
- 上下文感知，保持对话连贯性
- 情绪系统，女友会根据对话改变心情

### 💝 互动系统
- **亲亲** - 输入"亲亲"或"亲我"触发亲吻动画
- **摸头** - 输入"摸摸头"或"摸头"触发摸头动画  
- **睡觉** - 输入"躺下"或"睡觉"一起入睡
- **抱抱** - 输入"抱"或"抱抱"触发拥抱动作

### 🧠 记忆系统
- 自动记录对话历史
- 智能记忆压缩，将长对话总结为背景信息
- 长期记忆保存，保持角色一致性

### ⚙️ 配置界面
- 游戏内 GUI 配置 AI 参数
- 支持修改 API Key、API 地址和模型名称
- 配置自动保存到本地文件

## 🚀 新特性

- 新增模型配置功能，支持切换不同 AI 模型
- 修复配置加载逻辑，确保配置正确读取
- 支持通过环境变量配置 API 参数
- 优化对话响应速度

## 🔧 技术改进

- 修复 `AIConfig.java` 中配置加载的条件判断 bug
- 将硬编码的模型名称改为从配置读取
- 支持 `DEEPSEEK_MODEL` 环境变量
- 配置文件增加 `model` 字段

## 📖 使用说明

### 安装
1. 下载 `AI_Girl_Friend-1.0.0-1.20.1-forge.jar`
2. 将文件放入 `.minecraft/mods/` 目录
3. 启动游戏

### 配置
1. 使用命令 `/aiconfig` 打开配置界面
2. 输入你的 API Key
3. 选择模型（DeepSeek 或 GLM）
4. 点击保存配置

## 🐛 已知问题

- 首次配置后需要重启游戏才能完全生效
- 部分模型可能需要调整 API 地址格式
